# Student Management System

A full-stack web application to manage student records with authentication and dashboard.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

##  Features
- User Authentication (Login/Register)
- Add, Update, Delete Students
- View Student List
- Search and Filter Students
- Responsive UI
